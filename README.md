# הכספת 🔐
### הפלטפורמה של ג׳אוריוס

## מה יש כאן?
- דף כניסה ממותג
- דשבורד ראשי עם ניווט
- ספריית סרטונים: "מישרדות לחופש" + "שיעורי הכספת"
- צ׳אט בוט ג׳אוריוס (בסיס)
- ליווי אישי (אזור נעול)
- פאנל ניהול: הוספה/עריכה/מחיקה של סרטונים + ניהול משתמשים

## התקנה
```bash
npm install
npm run dev
```

## הגדרות
צור קובץ `.env.local` עם:
```
NEXT_PUBLIC_SUPABASE_URL=your-url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-key
```
